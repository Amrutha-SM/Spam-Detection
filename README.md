# Spam Detection Project

This project is a **Spam Detection System** built using **Python and Machine Learning**.  
It classifies a given message as **Spam** or **Not Spam (Ham)** based on learned patterns from training data.

---

## Features
- Detects spam messages accurately
- Uses machine learning classification techniques
- Simple and easy-to-run Python implementation
- Can be extended into a web or GUI application

---

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
